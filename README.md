# Library
LibraryApp with Flask&amp;SqlAlchemy&amp;Sqlite


### Installation
```
$ pip3 install -r ./requirements.txt
```
##### Inside the folder, run:
```
$ python book.py
```
##### Then Open *localhost:5000*
